Welcome.

Rakumarket is a Ruby abstraction of the Rakuten Web Service 楽天市場系API.

ItemSearch

Searches for items matching the given keyword.

item_list = Rakumarket.item_search "keyword", options

Request

Rakumarket.item_search takes a hash of request options, for example:

{:price => {:maximum => 30000}, :must_be_available => true}

ItemLookup

Finds an item by its item code.

item = Rakumarket.item_lookup "item_code", options

Request

Rakumarket.item_lookup takes a hash of request options, for example:

{:mobile => true}

Here is a full yaml representation of accepted keys and values for the options hash:

> puts Rakumarket::ItemLookupClient.inspect_options.to_yaml
---
:mobile: true|false

Response

Rakumarket.item_lookup returns a Rakumarket::Item object representing one item, for example:

GenreSearch

Returns genre information by genre id including parent and descendent genres.

genre = Rakumarket.genre_search "559887", options

Request

Rakumarket.genre_search takes a hash of request options, for example:

{:return_immediate_parent => true}

Here is a full yaml representation of accepted keys and values for the options hash:

> puts Rakumarket::GenreSearchClient.inspect_options.to_yaml
---
:return_immediate_parent: true|false

Response

Rakumarket.genre_search returns a Rakumarket::GenreFamily object representing a genre and its ascending and descending genres, for example:

genre.id
 => "559887"
genre.name
 => "500ml"
genre.parent.id
 => "559886"
genre.children.size
 => 5
genre.children.first.id
 => "559887"

Here is a full yaml representation of Item attributes and values:

> puts Rakumarket::GenreFamily.inspect_attributes.to_yaml
---
:id: ''
:name: ''
:parent:
  :id: ''
  :name: ''
:children:
- :id: ''
  :name: ''

ItemRanking

Returns a ranked list of top-selling items within a given genre and/or demographic.

item_list = Rakumarket.item_ranking options

Request

Rakumarket.item_ranking takes a hash of request options, for example:

{:age_range => (20..29), :sex => :female}

Here is a full yaml representation of accepted keys and values for the options hash:

> puts Rakumarket::ItemRankingClient.inspect_options.to_yaml
---
:return_immediate_parent: true|false

Response

Rakumarket.genre_search returns a Rakumarket::GenreFamily object representing a genre and its ascending and descending genres, for example:

genre.id
 => "559887"
genre.name
 => "500ml"
genre.parent.id
 => "559886"
genre.children.size
 => 5
genre.children.first.id
 => "559887"

See ItemSearch for a full representation of ItemList attributes and values.
