= Rakumarket

A reader-friendly Ruby abstraction of the Rakuten Market API.


== DESCRIPTION:

The goal was to abstract Rakuten's complex API away from users and provide a simpler, easier-to-read Ruby interface in its place.


== FEATURES:

* Idiomatic Ruby
* Simpler and more readable representation of the underlying API
* Returns Hashie::Mash[https://github.com/intridea/hashie]


== GOALS:

* Completeness


== CONFIGURATION:

  Rakumarket.developer_id = 'your_developer_id'

== INTERFACE:

=== Item Search

  Rakumarket.item_search "roomba", options

`options` is a hash which may include:

* :affiliate_id => 'affiliate_id'
* :call_back => 'call_back'
* :shop_code => 'shop_code'
* :genre_id => 'genre_id'
* :items_per_page => 30
* :page => 1
* :order => 'attribute asc|desc'
* :price => {:minimum => 0}
* :price => {:maximum => 10000}
* :must_be_available => false|true
* :deep_search => false|true
* :mobile => false|true
* :must_have_image => false
* :or_search => false|true
* :exclude_keyword => 'keyword'
* :include_genre_info => false|true
* :purchase_type => 'normal'|'regular'|'distribution'
* :shipping => {:must_ship_international => false|true}
* :shipping => {:country => 'country name'}
* :shipping => {:must_ship_next_day => false|true}
* :shipping => {:next_day_area => 'area name'}
* :shipping => {:cost_included => false|true}
* :must_have_point_multiplication => false|true
* :point_multiplication_factor => false|true
* :must_accept_credit_cards => false|true
